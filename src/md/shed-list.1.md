# NAME
shed-list - print list of code and name of available sheds.

# SYNOPSIS
**shed-list**

# DESCRIPTION
Print list of code and name of available sheds.

# EXAMPLES
**shed-list**\
\ \ \ \  Code\ \ \ \ \  Name\
\ \ \ \  D\ \ \ \ \ \ \ \  shed-D-development\
\ \ \ \  P\ \ \ \ \ \ \ \  shed-P-production\
\ \ \ \  T\ \ \ \ \ \ \ \  shed-T-testing
